# float_10bit
Save the float between 0.0001 and 1 with just 10 bits




將0.0001~1的數字轉為2進制10bit

指數位有3位數，精度位數有7位數

ex. 0.0034567

第一步驟：直接將數值轉為二進制
    0.0034567 => 0.00000000101 = 1.01 × 2^2
    指數位最大為7
    超過7以7表示

第二步驟：計算指數
    3 => 010

第三步驟：計算精度
    有七位數
    原各個精度代表0.5, 0.25, 0.125,.....
    因指數位為2，可得知第一位數出現在小數第二位
    因此此數僅記錄小數第三位數後7個bit
    即個精度代表0.125, 0.0625, 0.03125,.....

    因，第一位數記錄在指數位
    此處儲存0100000

最後結果即010 0100000


------------------反解



取得 010 0100000

第一步驟 計算指數
    010 => 2
    可得知此binary 開頭為0.01

第二步驟 計算精度
    0100000直接接續在0.01後
    可得知此binary為 0.010100000

第三步驟 轉換為十進制
    0.0101 = 0.0325
